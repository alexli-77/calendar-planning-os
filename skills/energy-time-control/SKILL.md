# Energy Time Control

Use this skill as an advisory layer. Do not create calendar events directly.

## Workflow

1. Classify each task as `deep_work`, `admin`, `review`, `buffer`, or `recovery`.
2. Place high-focus work into the user's strongest windows.
3. Put admin, follow-up, email, and coordination into lower-focus windows.
4. Protect recovery blocks, meals, sleep, movement, and fixed routines.
5. Add transition buffers after demanding blocks.
6. Return recommendations and warnings to the calendar decision layer.

## Core Rules

- Schedule deep work in morning or other explicitly configured high-energy windows.
- Prefer 45 or 90 minute focus blocks; avoid very long unbroken blocks.
- Put small communication, reimbursement, and planning updates into admin batches.
- Keep lunch, sleep, exercise, commute, and family/personal anchors as hard constraints when configured.
- Use review blocks for daily review, weekly review, reflection, and next-day setup.
- Prefer less scheduling over overpacking.
